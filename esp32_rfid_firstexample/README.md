# Project RFID Firstexample
The goal is to successfully bind a RFID-RC522 module to a Wemos Lolin32 and read RFID tags. More information you can find:
* http://www.esp32learning.com/code/esp32-and-rfid-rc522-module-example.php
* https://www.instructables.com/id/ESP32-With-RFID-Access-Control/
